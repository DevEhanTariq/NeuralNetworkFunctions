# How To Run:

Type all of these commands into your terminal

### Step 1:

- `cd rusty_snek`

### Step 2:

- `source .venv/bin/activate`

### step 3:

- Try both:
- `python main.py`
- `python3 main.py`

### IMPORTANT:

- For every change in rust code, run `maturin develop`, with steps 1 and 2 complete.
